# MNIST Digit Recognition

This project demonstrates training Dense and Convolutional Neural Networks on the MNIST dataset using Keras and TensorFlow.

**Models:**
- Dense Neural Network (1 hidden layer, dropout 0.3)
- Convolutional Neural Network (2 conv layers + pooling + dropout 0.4)

**Results:**
- Dense NN accuracy: ~97%
- CNN accuracy: ~99%

Tools: Python, NumPy, TensorFlow, Keras
